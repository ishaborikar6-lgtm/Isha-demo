# User Management System

A web-based User Management System developed using HTML, CSS and JavaScript.

## Features

- Add New User
- View User Details
- Edit User Information
- Delete User with Confirmation Popup
- Search Users by Name or Email
- Store Unlimited User Records
- User-Friendly Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Project Overview

This project allows users to manage records efficiently. Users can add, search, view, edit and delete records through an interactive interface.
